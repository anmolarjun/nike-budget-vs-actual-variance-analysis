# Nike Budget vs Actual Variance Analysis

## Overview

This project presents a Budget vs Actual Variance Analysis for Nike using Microsoft Excel. It compares planned financial performance against actual results and highlights key variances to evaluate business performance.

## Project Objectives

- Compare budgeted and actual financial figures
- Calculate absolute and percentage variances
- Identify favorable and unfavorable performance
- Demonstrate financial planning and performance analysis skills

## Skills Demonstrated

- Financial Analysis
- Variance Analysis
- Budgeting & Forecasting
- Microsoft Excel
- Financial Modeling
- Data Interpretation
- Business Performance Analysis

## Tools Used

- Microsoft Excel

## Key Learning Outcomes

- Measuring operational performance against budget
- Identifying favorable and unfavorable variances
- Understanding drivers of financial performance
- Presenting financial information in a structured format
